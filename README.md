# zhoudian 周店
此应用基于web互联网思维为线下到店就餐的顾客在线上提供餐馆内真实餐馆信息、菜单信息，降低点餐难度，为店家提供新的宣传渠道，提升经营效率。

## 技术选型
### 前端
vue+elementUI

### api sdk
腾讯云开发sdk

### 后端
无

### 数据库
腾讯云开发数据库

### 图片存储
腾讯云开发云存储（支持cdn）

### 静态资源托管
腾讯云开发云存储（支持cdn）

### web服务器
GithubPages

### 定位服务
H5 GEO api

### 图片审核
人工判别系统
